# Collier Township Project Tracker

Track and follow township policy proposals and community projects in Collier Township.

**Live:** https://my-app-xi-nine-22.vercel.app/

---

## Web App (Next.js)

```bash
cd my-app
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

---

## Mobile App (React Native / Expo)

### Prerequisites

- [Expo Go](https://apps.apple.com/app/expo-go/id982107779) installed on your iOS device
- Your phone and computer on the same Wi-Fi network

### Run

```bash
cd mobile
npm install
npx expo start
```

Scan the QR code in the terminal with your iPhone camera — it will open in Expo Go.

> **Tip:** If the QR code doesn't work, press `s` in the terminal to switch to Expo Go mode, then try again.
